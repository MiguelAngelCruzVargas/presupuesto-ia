# 📋 Guía: Exportar Catálogo desde CONSTRUBASE-LIBRE

## 🎯 Página Correcta

La página que estás viendo (`neodata.mx/productos`) es para descargar **software**, no para exportar catálogos de precios.

Para exportar el **catálogo de precios**, necesitas acceder a **CONSTRUBASE-LIBRE**.

## 🔗 Acceso Correcto

### Opción 1: Desde neodatapu.appspot.com (Consulta de Presupuesto)

1. **Ir a**: https://neodatapu.appspot.com
2. **Iniciar sesión** con: vargasmiguelangelc@gmail.com
3. **Buscar sección** de "Catálogo" o "Consultas"

### Opción 2: Desde Neodata PU (Software)

Si tienes instalado **Neodata PU Win+** (v25.5.0 que viste en la página):

1. **Abrir el software** Neodata PU Win+
2. **Buscar opción**:
   - "Exportar Catálogo"
   - "Exportar Precios"
   - "Copiar Catálogo"
   - Menú: Archivo → Exportar

## 📊 Cómo Exportar desde el Software Neodata PU

### Método 1: Desde el Software Desktop

Si tienes acceso al software Neodata PU Win+:

1. **Abrir** Neodata PU Win+ v25.5.0
2. **Ir a**: Catálogos o Precios Unitarios
3. **Buscar opción de exportar**:
   - Archivo → Exportar → Excel/CSV
   - O botón "Exportar Catálogo"
   - O "Copiar Catálogo" y pegar en Excel

### Método 2: Desde la Web (neodatapu.appspot.com)

1. **Acceder** a: https://neodatapu.appspot.com
2. **Iniciar sesión**
3. En la interfaz de "M-Consulta de Presupuesto":
   - **Seleccionar** todas las partidas (8,181)
   - **Copiar** la tabla completa (Ctrl+A, Ctrl+C)
   - **Pegar** en Excel (Ctrl+V)
   - **Guardar** como CSV

## 🎯 Alternativa: Crear Script de Extracción
